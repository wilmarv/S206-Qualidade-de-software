# S206 L1 Trabalho Pratico
Teste de UI (User Interface) com cypress 

### 🔧 Instalação

#### Baixar dependencia do projeto.

Terminal do diretorio do projeto execute:

```
npm install
```
ou
```
yarn
```

### ⚙️ Inicializando o CyperPress

```
yarn testYt
```
### ⚙️ Executando specs por linha de comando

```
yarn spec
```
